# Daybreak Onboarding Frontend (Next.js)

Next.js (Pages router) frontend for the Daybreak Parent Onboarding Microsite.

## Prerequisites
- Node.js 18+
- npm

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
# Visit http://localhost:3000
```

## Scripts
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm start` — run production server
- `npm run lint` — lint with ESLint
- `npm run type-check` — TypeScript type checking
- `npm test` — placeholder test command

## Notes
- Uses TypeScript, ESLint, Prettier, and Tailwind (v4) by default.
